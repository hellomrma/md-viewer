# 샘플 문서

이 파일은 `public/docs/`에 둔 마크다운이 라이브러리에 자동으로 나타나는지 확인하기 위한 샘플입니다.

## 동작 방식

- `public/docs/` 폴더의 `.md` / `.markdown` 파일이 자동으로 인식됩니다.
- 빌드 시 또는 dev 서버 실행 시 `manifest.json`이 자동으로 생성됩니다.
- 카드를 클릭하면 URL이 `?doc=파일명.md`로 갱신되어 공유 가능합니다.

## 사용 방법

1. `public/docs/` 폴더에 `.md` 파일을 둡니다.
2. dev 서버가 자동으로 감지하고 페이지를 새로고침합니다.
3. 메인 페이지의 Library 섹션에서 문서를 선택합니다.

```ts
console.log("코드 블록도 정상 렌더링됩니다.");
```
